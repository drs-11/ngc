## Commands 

Initialise a repository:

```
$ ngc init
```

Commit a repo:

```
$ ngc commit
```

Get status of modified files:

```
$ ngc status
```

Reset to the previous commit:

```
$ ngc reset
```

Get the logs of commits:

```
$ ngc log
```

Checkout a specific commit:

```
$ ngc checkout
```
